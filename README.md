# qiancun
A react-native app.

## get & run

```shell
$ git clone https://github.com/Txiaozhe/qiancun.git
```
```shell
$ cd qiancun
```
```shell
$ npm install
```
```shell
$ react-native run-android         #for android
$ react-native run-ios             #for ios simulator
$ react-native run-ios --device    #for ios device
```
